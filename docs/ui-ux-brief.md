# Becoming — UI/UX Brief

## 1. Design Goal

Becoming should feel like a premium consumer startup product, not a generic habit tracker.

The interface should be:

- Calm
- Premium
- Cinematic
- Emotional
- Modern
- Reflective
- Motivating without guilt

The design should make users feel like they are entering a personal growth space, not a productivity spreadsheet.

## 2. Brand Feeling

Becoming should feel like:

- A personal operating system
- A life dashboard
- A seasonal journal
- A progress tracker
- A Wrapped-style reflection product
- A calm wellness companion

It should not feel like:

- A corporate SaaS dashboard
- A childish habit tracker
- A Notion template clone
- A medical app
- A productivity guilt machine
- A generic AI-generated landing page

## 3. Core Brand Language

### App Name

Becoming

### Tagline

Track your becoming.

### Hero Line

Become the person you said you would.

### Onboarding Question

Who are you becoming this year?

### Dashboard Name

Today’s Becoming

### Season Recap

This Season’s Becoming

### Year Recap

Your Year of Becoming

### Final Recap Message

Look how far you’ve come. See who you’re becoming.

## 4. Visual Direction

### Style

- Dark premium interface
- Soft gradients
- Glass cards
- Gem-inspired glow
- Subtle borders
- Large story cards
- Clean dashboard layout
- Generous spacing
- Strong typography hierarchy

### Mood

- Reflective
- Focused
- Hopeful
- Calm
- Slightly cinematic

## 5. Color System

Use a dark base with gem-inspired accents.

### Base Colors

- Deep navy
- Near-black purple
- Soft charcoal
- Muted slate

### Accent Colors

- Amethyst purple
- Emerald green
- Sapphire blue
- Rose quartz pink
- Gold highlight

### Usage

- Purple: primary brand
- Emerald: positive progress
- Gold: achievements
- Blue: focus and calm
- Rose: mood/reflection

## 6. Typography

Use clean, modern typography.

Recommended options:

- Inter
- Geist
- Manrope
- Satoshi if available

### Typography Rules

- Big emotional headings
- Short paragraphs
- Dashboard labels should be small and clear
- Numbers should feel prominent
- Recap pages should use large cinematic text

## 7. Layout Principles

### Landing Page

Should feel emotional and premium.

Must include:

- Strong hero
- Dashboard preview
- Explanation sections
- CTA buttons
- Social/startup polish

### Dashboard

Should feel like a control center.

Layout:

- Top header with current season
- Main score area
- Dashboard card grid
- Right-side focus/reflection panel
- Heatmap section
- Progress chart section

### Daily Check-In

Should feel quick and calming.

Use:

- Checklist cards
- Sliders
- Simple inputs
- Encouraging confirmation state

### Wrapped Page

Should feel cinematic.

Use:

- Big story cards
- Large numbers
- Gradient backgrounds
- Animated reveal if possible
- Share button
